fait un code VHDL commantee et base sur des machines a etats pour faire une IHM module gestion des boutons poussoirs avec:
entre des bouttons poussoir: BP_Babord,BP_Tribord,BP_STBY, un clk_50M,
des sorties led ledBabord (sur 1 bits), ledTribord (sur 1 bits), ledSTBY (sur 1 bits), out_bip (sur 1 bits) et un code fonction pour chaque etat du fonctionnemnt (sur 4 bits)

le fonctionnement de notre code est le suivents:
pour la gestion des BP_Babord,BP_Tribord,BP_STBY: avec en sortie le code fonction

pour passee du mode auto au mode manuel, un simple appuit sur le BP_STBY, en mode manuel code fonction sur "0000" et en mode auto "0011"
dans le mode manuel :un appuit simple sur BP_Babord et code fonction "0001"
dans le mode manuel :un appuit simple sur BP_Tribord "0010"
apres chaque front decendant d'un appuit en revien sur le mode manuel "0000"

dans le mode auto "0011": on une une etat tempo pour detectee le temps d'appuit sur BP_Babord et BP_Tribord
si on a un appuit simple sur BP_Babord "0100"
si on a un appuit simple sur BP_Tribord "0111"
si on a un appuit long sur BP_Babord "0101"
si on a un appuit long sur BP_Tribord "0110"
apres chaque front decendant d'un appuit long ou simple en revien sur le mode auto
fin de fonctionnement.

pour un appuit simple: c'est un appuit moins de 1s
pour un appuit long: c'est appuit supperieur a 3s

pour la gestion des leds et bipeur: led ledBabord (sur 1 bits), ledTribord (sur 1 bits), ledSTBY (sur 1 bits), out_bip (sur 1 bits)
pour chaque code fonction on a: 

-- =0000: mode manuel et led ledSTBY et clignote toujour chaque 1s et out_bip clignote une suel fois
-- =0001: led ledBabord et et out_bip clignote une suel fois 
-- =0010: ledTribord et et out_bip clignote une suel fois
-- =0011:mode auto et led ledSTBY reste allumee toujour et out_bip clignote une suel fois
-- =0100: led ledBabord et out_bip clignote une suel fois 
-- =0101: led ledBabord et out_bip clignote une deux fois 
-- =0111: ledTribord et out_bip clignote une suel fois
-- =0110: ledTribord et out_bip clignote une deux fois





deux mode de fonctionnements:

en mode auto : un apput simple sur BP_STBY : code fonction '0011' et ledSTBY allumee et ilya un simple bip de out_bip, les autres leds sont etainte
en mode auto : un apput long sur BP_STBY : code fonction '0011' et ledSTBY allumee et clingote et ilya un double bip de out_bip, les autres les sont etande on peut en ce mode auto (seul avec l'appuit long) de controlee les autres entre, pour :
un apuit simple sur BP_Babord : code fonction : 0100 et led ledBabord s'allume une fois et ilya un simple bip de out_bip
un apuit simple sur BP_Tribord : code fonction : 0111 et led ledTribord s'allume une fois et ilya un simple bip de out_bip
un apuit long sur BP_Babord : code fonction : 0101 et led ledBabord s'allume et clignote jusqua relache et ilya une double bip de out_bip
un apuit long sur BP_Tribord : code fonction : 0110 et led ledTribord s'allume et clignote jusqua relache et ilya une double bip de out_bip


pour passe du mode auto au mode manuel cap: un apput simple sur BP_STBY : code fonction '0000' et ledSTBY allumee et clignote et ilya un simple bip de out_bip, les autres leds sont etainte 
en mode manuel:

un apuit simple sur BP_Babord : code fonction : 0001 et led ledBabord s'allume une fois et ilya un simple bip de out_bip
un apuit simple sur BP_Tribord : code fonction : 0010 et led ledTribord s'allume une fois et ilya un simple bip de out_bip

des pressisions par rapport au code:
l'etat initiale du systeme est en mode manuel avec la led ledSTBY allumee, clignote et code fonction '0000'.
pour le fonctionnement il faut geree des temporisateur pour les leds et les appuis et le code fonction sera une sortie sur 4 bits 

la carte a un horloge de clk_50M soir l'entre de notre code aussi

un appuit courte ou simple appuit est un apuis inferireur à 1s
un un appuit long est un appuit superieur à un 1s
les leds clignote avec un cycle de 1s aussi
-- valeurs de codeFonction:
-- =0000: pas d'action, le pilote est en veille
-- =0001: mode manuel action vérin babord
-- =0010: mode manuel action vérin tribord
-- =0011: mode pilote automatique/cap
-- =0100: incrément de 1° consigne de cap
-- =0101: incrément de 10° consigne de cap
-- =0111: décrément de 1° consigne de cap
-- =0110: décrément de 10° consigne de cap




États:
- Manual_Mode: État initial où le système est en mode manuel.
- Auto_Mode: Le système est en mode automatique.
- Auto_Long_Press: Appui long sur BP_STBY en mode automatique.
- Auto_Babord: Appui simple sur BP_Babord en mode automatique.
- Auto_Tribord: Appui simple sur BP_Tribord en mode automatique.
- Auto_Babord_Long: Appui long sur BP_Babord en mode automatique.
- Auto_Tribord_Long: Appui long sur BP_Tribord en mode automatique.

Transitions:
1. Manual_Mode -> Auto_Mode (BP_STBY pressé)
2. Manual_Mode -> Auto_Babord (BP_Babord pressé)
3. Manual_Mode -> Auto_Tribord (BP_Tribord pressé)

4. Auto_Mode -> Manual_Mode (BP_STBY pressé)
5. Auto_Mode -> Auto_Long_Press (BP_STBY pressé longtemps)
6. Auto_Mode -> Auto_Babord (BP_Babord pressé)
7. Auto_Mode -> Auto_Tribord (BP_Tribord pressé)

8. Auto_Long_Press -> Manual_Mode (BP_STBY relâché)
9. Auto_Long_Press -> Auto_Mode (BP_STBY pressé)

10. Auto_Babord -> Manual_Mode (BP_STBY pressé)
11. Auto_Babord -> Auto_Babord_Long (BP_Babord pressé longtemps)
12. Auto_Babord -> Auto_Mode (BP_Babord relâché)

13. Auto_Tribord -> Manual_Mode (BP_STBY pressé)
14. Auto_Tribord -> Auto_Tribord_Long (BP_Tribord pressé longtemps)
15. Auto_Tribord -> Auto_Mode (BP_Tribord relâché)

16. Auto_Babord_Long -> Manual_Mode (BP_STBY pressé)
17. Auto_Babord_Long -> Auto_Mode (BP_Babord relâché)

18. Auto_Tribord_Long -> Manual_Mode (BP_STBY pressé)
19. Auto_Tribord_Long -> Auto_Mode (BP_Tribord relâché)

Actions:
- CodeFonction = "0000", ledSTBY allumée et clignotante (Manual_Mode)
- CodeFonction = "0011", ledSTBY allumée, simple bip (Auto_Mode, Auto_Long_Press)
- CodeFonction = "0100", ledBabord allumée une fois, simple bip (Auto_Babord)
- CodeFonction = "0111", ledTribord allumée une fois, simple bip (Auto_Tribord)
- CodeFonction = "0101", ledBabord allumée et clignotante, double bip (Auto_Babord_Long)
- CodeFonction = "0110", ledTribord allumée et clignotante, double bip (Auto_Tribord_Long)
